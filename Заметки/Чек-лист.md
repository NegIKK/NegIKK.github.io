# LOD

Кидание на проверку:
Собираем финальные фбх, ворлд и обычную. Собираем весь архив. Только после этого собираем в спп. После чего кидаем финальный архив и спп

## Нейминг
Не делаем нижнее подчеркивание у нумерации. Stock01, Stock02

- FBX — Имя проекта вместе с номером патча
- Материалы - Имя проекта без номера + суффиксы
	- `_mat`
	- `_att_mat`
	- `_world_mat`
	- `_att_world_mat`
- Текстуры — Имя проекта без номера патча + суффикс текстурной карты + `_World`
