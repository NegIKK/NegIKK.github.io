TrustTunnel Client. Гайд

Альтернативное подключение к серваку. Иногда работает лучше чем наше старое. Запасной вариант на всякий случай. Я иногда пользуюсь уже

Качаем прогу
Android
Apple

В начале создадим профиль с правилами для нужных нам адресов

  • Запускаем прогу и снизу переходим в Routing
  • Нажимаем Add Profile
  • Переходим во вкладку VPN
  • В текстовое окно вставляем нужные нам домены и IP Список доменов для ВПН
  • Потом жмем на настройки, выбираем Change default routing
  • Выбираем Bypass
  • Не забываем прожимать Save
  • Созданный профиль можно назвать, нажав Edit Profile. У меня он называется Selective

Теперь делаем подключение к серверу
Пункты по порядку

Server name:
[Любое имя]

IP:
46.32.187.9:443

Domain:
matus.isgood.host

Username:
[Ваш логин]

Password:
[Ваш пароль]

Protocol:
HTTP/2

Routing profile:
[Выбираем профиль, который мы создали ранее]

DNS:
1.1.1.1

Если куда-то не подключается то тупо дописываем сайт в список example.com без www, http// и прочего такого. Иногда этого недостаточно, для больших сервисов, тогда тупо гуглим чет типа "domain list telegram", пример с телегой. Обычно первая или вторая ссылка содержат все что нам нужно
